# Dynamic Array

**Dynamic array** is a typed, growable array backed by the arena. No `malloc`/`free`, no separate bookkeeping.

## API

### Convention

A dynamic array is any struct with three fields named exactly `items`, `count`, and `capacity`:

```c
typedef struct {
  T *items; // pointer to element storage
  size_t count; // number of elements currently held
  size_t capacity; // allocated capacity in elements
} TArray;
```

Initialize it to all-zeros before use:

```c
TArray arr = {0};
```

The macros read and write these three fields by name, so the struct definition is all that is needed. No registration or constructor call.

### `ECEWO_DA_APPEND`

```c
ECEWO_DA_APPEND(ecewo_arena_t *arena, da *arr, T item);
```

Appends a single `item` to the dynamic array `arr`, growing the backing storage if needed.

Growth strategy: starts at `ARENA_DA_INIT_CAP` (default `256`) elements and doubles on each resize. All storage comes from arena; nothing is freed individually.

**Parameters**

| Parameter | Description                                   |
|-----------|-----------------------------------------------|
| `arena`   | Arena to allocate from                        |
| `arr`     | Pointer to the dynamic array struct           |
| `item`    | Value to append (must match the element type) |

**Example**

```c
#include "ecewo.h"

typedef struct {
  int *items;
  size_t count;
  size_t capacity;
} IntArray;

void example(ecewo_request_t *req, ecewo_response_t *res) {
  IntArray nums = {0};

  ECEWO_DA_APPEND(req->arena, &nums, 10);
  ECEWO_DA_APPEND(req->arena, &nums, 20);
  ECEWO_DA_APPEND(req->arena, &nums, 30);

  // nums.items = [10, 20, 30], nums.count = 3
}
```

### ECEWO_DA_APPEND_MANY

```c
ECEWO_DA_APPEND_MANY(ecewo_arena_t *arena, da *arr, T *new_items, size_t new_items_count);
```

Appends `new_items_count` elements from `new_items` to the dynamic array in a single `memcpy`. More efficient than calling `ECEWO_DA_APPEND` in a loop when the source is already a contiguous array.

**Parameters**

| Parameter         | Description                               |
|-------------------|-------------------------------------------|
| `arena`           | Arena to allocate from                    |
| `arr`             | Pointer to the dynamic array struct       |
| `new_items`       | Pointer to the first element to copy from |
| `new_items_count` | Number of elements to copy                |

**Example**

```c
#include "ecewo.h"

typedef struct {
  int *items;
  size_t count;
  size_t capacity;
} IntArray;

void example(ecewo_request_t *req, ecewo_response_t *res) {
  int src[] = {1, 2, 3, 4, 5};

  IntArray nums = {0};
  ECEWO_DA_APPEND_MANY(req->arena, &nums, src, 5);

  // nums.items = [1, 2, 3, 4, 5], nums.count = 5
}
```

## Examples

### Filtering rows into a dynamic array

You query all orders but only want to return the ones with `status = "pending"`. The count is unknown until you iterate.

**Without dynamic array**; two passes required:

```c
#include "ecewo.h"

void get_pending_orders(ecewo_request_t *req, ecewo_response_t *res) {
  DbResult *rows = db_query("SELECT id, customer, total, status FROM orders");

  // First pass: count matching rows
  int pending_count = 0;
  for (int i = 0; i < rows->count; i++) {
    if (strcmp(rows->data[i].status, "pending") == 0)
      pending_count++;
  }

  // Allocate exact size
  Order *pending = ecewo_alloc(req->arena, sizeof(Order) * pending_count);

  // Second pass: fill the array
  int idx = 0;
  for (int i = 0; i < rows->count; i++) {
    if (strcmp(rows->data[i].status, "pending") == 0) {
      pending[idx].id = rows->data[i].id;
      pending[idx].customer = rows->data[i].customer;
      pending[idx].total = rows->data[i].total;
      idx++;
    }
  }

  // ... serialize pending[0..pending_count]
}
```

**With `ECEWO_DA_APPEND`**; one pass:

```c
#include "ecewo.h"

typedef struct {
  Order  *items;
  size_t  count;
  size_t  capacity;
} OrderArray;

void get_pending_orders(ecewo_request_t *req, ecewo_response_t *res) {
  DbResult *rows = db_query("SELECT id, customer, total, status FROM orders");

  OrderArray pending = {0};

  for (int i = 0; i < rows->count; i++) {
    if (strcmp(rows->data[i].status, "pending") == 0) {
      Order o = {
        .id = rows->data[i].id,
        .customer = rows->data[i].customer,
        .total = rows->data[i].total,
      };
      ECEWO_DA_APPEND(req->arena, &pending, o);
    }
  }

  // ... serialize pending.items[0..pending.count]
}
```

### Merging two result sets

You query staff and contractors separately and want to return them as one combined list.

**Without `ECEWO_DA_APPEND_MANY`**; two loops:

```c
#include "ecewo.h"

typedef struct {
  Person *items;
  size_t  count;
  size_t  capacity;
} PersonArray;

void get_all_people(ecewo_request_t *req, ecewo_response_t *res) {
  DbResult *staff = db_query("SELECT name, role FROM staff");
  DbResult *contractors = db_query("SELECT name, role FROM contractors");

  PersonArray all = {0};

  for (int i = 0; i < staff->count; i++)
    ECEWO_DA_APPEND(req->arena, &all, (Person){ staff->data[i].name, staff->data[i].role });

  for (int i = 0; i < contractors->count; i++)
    ECEWO_DA_APPEND(req->arena, &all, (Person){ contractors->data[i].name, contractors->data[i].role });

  // ... serialize all.items[0..all.count]
}
```

**With `ECEWO_DA_APPEND_MANY`**; two lines, one `memcpy` per batch:

```c
#include "ecewo.h"

typedef struct {
  Person *items;
  size_t  count;
  size_t  capacity;
} PersonArray;

void get_all_people(ecewo_request_t *req, ecewo_response_t *res) {
  DbResult *staff = db_query("SELECT name, role FROM staff");
  DbResult *contractors = db_query("SELECT name, role FROM contractors");

  PersonArray all = {0};

  ECEWO_DA_APPEND_MANY(req->arena, &all, (Person *)staff->data, staff->count);
  ECEWO_DA_APPEND_MANY(req->arena, &all, (Person *)contractors->data, contractors->count);

  // ... serialize all.items[0..all.count]
}
```
